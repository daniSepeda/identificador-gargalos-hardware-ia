# 🖥️ Identificador de Gargalos e Recomendador de Atualização de Peças de Hardware

Projeto final destinado à disciplina de **Inteligência Artificial**, ministrada pelo Professor **Adolfo Colares**, na **Universidade Federal do Amapá (UNIFAP)**, no semestre **2026.1**.

## 👥 Componentes do Grupo

- Daniela Sepeda
- Geovanni Rodrigues
- Maria Fernanda Fernandes
- Miguel Peres

## 🎯 Objetivo do Projeto

Criar um sistema que possibilite a identificação de gargalos em tempo real no funcionamento de hardware, além de sugerir melhorias relacionadas aos componentes do computador, de acordo com a performance observada e os objetivos do sistema analisado.
